# 渐进记号

O(1) < O(lgn) < O(n) < O(nlgn) < O(n^2) < O(n^3) < O(2^n) < O(n!) < O(n^n)





Θ、O、Ω、ο、ω


|   渐进记号   |        粗略理解        |                          |
| :-------------: | :----------------------: | -------------------------- |
| f(x)=Θ(g(n)) | a\*g(n)≤f(x)≤b\*g(n) |                          |
| f(x)=O(g(n)) |     f(x)≤b\*g(n)     | 上界，一般用于最坏的情况 |
| f(x)=Ω(g(n)) |      a*g(n)≤f(x)      | 下界，一般用于最好的情况 |
| f(x)=ο(g(n)) |      f(x)＜b*g(n)      |                          |
| f(x)=ω(g(n)) |      a*g(n)＜f(x)      |                          |
